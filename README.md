# 🚗 Дорожный охранник — Telegram бот для водителей России

Telegram бот, который показывает камеры ГИБДД, парковки, пробки и зоны эвакуации в **любом городе России**. Работает по геопозиции.

## 📱 Возможности

| Функция | Описание |
|---------|----------|
| 📸 Камеры | Показывает камеры в радиусе 500 м (стационарные и передвижные) |
| 🅿️ Парковки | Бесплатные и платные парковки с количеством свободных мест |
| 🚗 Пробки | Текущая загруженность дорог, пробки на МКАД/ТТК |
| 🚨 Эвакуаторы | Зоны активной эвакуации, контакты штрафстоянок |
| ➕ Добавление | Пользователи могут добавлять новые камеры |
| 📢 Репорты | Сообщения об ошибках и новых объектах |

## 🏙️ Поддерживаемые города

Бот автоматически определяет город по геопозиции. Данные есть для:

- Москва
- Санкт-Петербург
- Казань
- Екатеринбург
- Новосибирск
- (и все остальные — база пополняется пользователями)
